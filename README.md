# Behavioural Biases in Finance

This site presents a number of behavioural biases in financial decision-making, as those are documented in the paper “The effect of behavioral biases on financial decisions”, by David Peon and Manel Antelo.

The aim is to offer a reference tool for students, practitioners, and anyone curious about the psychology behind economic choices. Each bias presents its definition, one or more examples associated with it, as well as cross-links to related biases.

It was coded using Quarto and Python.

Live version: https://akprodromou.github.io/behavioural-biases/
